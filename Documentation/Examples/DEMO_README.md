# PoolMaster Demo Scene

**Professional showcase of real-world pooling use-cases**

## Quick Start

1. Open `Assets/PoolMaster/Examples/DemoScene.unity`
2. Press **Play**
3. Use **NEXT ►** and **◄ PREVIOUS** buttons to navigate
4. Each demo runs independently - no performance issues!

## What You'll See

**7 practical pooling scenarios** solving real game development problems:

### 1. Projectile Pooling *(Most Common)*
- Bullets, arrows, missiles - the #1 pooling use-case
- High-frequency spawning (multiple/second)
- Perfect for: Shooting games, tower defense, action

### 2. VFX & Particle Pooling
- Explosions, impacts, collectibles
- Event-driven visual effects
- Perfect for: Particle systems, UI effects, polish

### 3. Enemy & NPC Pooling  
- Wave spawning, enemy groups
- Batch operations (5-10 at once)
- Perfect for: Wave spawners, AI, traffic systems

### 4. Particle Burst Effects
- 20-50+ particles instantly  
- Dynamic pool expansion
- Perfect for: Explosions, confetti, debris

### 5. Basic Object Pooling
- Simple spawn/despawn foundation
- Automatic initialization
- Perfect for: Pickups, platforms, UI, any GameObject

### 6. Thread-Safe Command Buffer *(Advanced)*
- Queue spawns from background threads
- Deferred execution
- Perfect for: Procedural generation, async workflows

### 7. Collection Pooling *(Zero Allocation)*
- Reuse List/Dictionary/HashSet
- Eliminate garbage collection
- Perfect for: Data processing, temporary buffers
